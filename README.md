# subdee's App Store

This repository is my personal custom app store for [Runtipi](https://runtipi.io) 

## Apps available

- [AdventureLog](https://github.com/seanmorley15/AdventureLog) - The ultimate travel companion
- [Calibre-Web Automated](https://github.com/crocodilestick/Calibre-Web-Automated) - All-in-one self-hosted digital library
- [Endurain](https://docs.endurain.com/) - Self-hosted fitness tracking service
- [Papra](https://github.com/papra-hq/papra) - Minimalistic document archiving platform
- [TREK](https://github.com/liketrek/TREK) - Self-hosted travel and trip planner
- [Wanderer](https://wanderer.to/) - Trail database
- [Wishlist](https://github.com/cmintey/wishlist) - A sharable wishlist for friends and family
- [Zerobyte](https://github.com/nicotsx/zerobyte) - Powerful backup automation for your remote storage

## Installation

You can use this app store following these steps:

1. In the Runtipi dashboard, go to Settings and then App Stores
2. Click Add App Store and give a name for the store and for the URL use `https://github.com/subdee/tipi-store`.
3. Now all of the above apps are available to install
