## About

Wishlist is a self-hosted wishlist application that you can share with your friends and family. You no longer have to
wonder what to get your parents for the holidays, simply check their wishlist and claim any available item. With a
simple user interface, even the grandparents can get involved!

## Features

- [x] Claim items on a wishlist
- [x] Check off claimed items as purchased
- [x] Automatically fetch product data from URL
- [x] Invite users via email (SMTP configuration required)
- [x] Options for [suggestions](#suggestions)
- [x] PWA Support
- [x] Multiple groups
- [x] Registry Mode (single list)
- [x] OAuth Authentication
